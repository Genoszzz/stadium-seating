# Stadium Seating

一个面向球场、会场、活动现场的可视化排座与 TIFO 图样生成工具。

项目是纯前端单页应用，适合快速搭建座位区域、删除不参与展示的座位、输入英文字母生成占座图样，并导出现场布置所需的图片和座位清单。

## 功能

- **座位区编辑**
  - 添加多个座位区域
  - 编辑行数、列数、座位尺寸、间距、旋转角度、弧线强度
  - 拖拽移动座位区，自动吸附网格

- **座位操作**
  - 单击座位切换占座状态
  - 右击删除座位，右击已删除位置恢复座位
  - `Shift + 单击` 多选座位
  - `Shift + 拖拽` 框选座位
  - 固定座位，避免被误操作

- **字母图样生成**
  - 输入英文后自动生成占座图样
  - 删除座位后，会在剩余可用座位中寻找合适的完整区域重新生成字母
  - 避免在扁长区域里过度拉伸字母

- **导入与导出**
  - 导出整张画布 PNG
  - 导出单个座位区高清 PNG
  - 导出/导入 MD 数据文件，用于保存和恢复布局
  - 导出 TIFO 占座清单 CSV，包含座位区域、行、列、座位编号、区域内序号

## 快捷操作

| 操作 | 说明 |
| --- | --- |
| 单击座位 | 切换占座状态 |
| 右击座位 | 删除座位 |
| 右击已删除座位 | 恢复座位 |
| Shift + 单击 | 多选座位 |
| Shift + 拖拽空白区域 | 框选座位 |
| 拖拽空白区域 | 平移画布 |
| 滚轮 | 缩放画布 |
| Space | 切换选中座位占座状态 |
| F | 固定/取消固定选中座位 |
| Delete | 删除选中座位 |
| Escape | 取消选中 |

## 本地运行

环境要求：

- Node.js 20 或更高版本
- npm

安装依赖：

```bash
npm install
```

启动开发服务器：

```bash
npm run dev -- --host 127.0.0.1
```

浏览器打开：

```text
http://127.0.0.1:5173/
```

生产构建：

```bash
npm run build
```

预览构建结果：

```bash
npm run preview
```

## 数据说明

项目目前不依赖后端服务，所有编辑状态保存在浏览器内存中。

如需长期保存布局，请使用工具栏的 **画布MD** 导出数据文件；下次使用时通过 **导入** 恢复。

## 适用场景
